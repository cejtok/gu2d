Add a box to the physics system, and returns the index of box added.

Parameters
xBox x-position.
yBox y-position.
wBox width.
hBox width.
dynamicGU_BOX_STATIC,GU_BOX_DYNAMIC.
gravityGravity acceleration.
vclampMaximun speed.

Use GU_BOX_STATIC when you don't want movement to be affected by colliding with the other boxes. Use GU_BOX_DYNAMIC when you want the movement to be affected by the other boxes. The sign of the return value is negative when GU_BOX_STATIC is used, otherwise it is positive.

void guDrawPhysics ( void  )

Draw the boxes.

When the world context is established (guEnableWorld()) the coordinates will be taken on the world axes, and the drawing can be viewed depending on the position of the camera. If the world context is not set or is disabled (guDisableWorld()), the position coordinates are taken on the axes of the screen.

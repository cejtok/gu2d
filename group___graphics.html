void guLoadTexture ( char *  filename,
int *  texture 
)

Loads a texture from game memory (Assets), not from disk.

Parameters
filenamefile name of the texture file just as it appears in the "textures" folder, for example "wall.png".
[out]texturethe texture address.

The dimensions of a texture must be power of 2, the maximum allowed is 512x512. It is recommended to use dimensions less than or equal to 256x256, and the width does not need to be equal to the height.

int tex_box;
guLoadTexture("box.png", &tex_box);

void guSetAnimation ( GUAnimator *  A,
int  animation 
)

Sets the current animation to be run by the Animator.

Parameters
APointer to the Animator.
animationAnimation index, which returned by guAddAnimation().

When the current animation is different from the one that will be established, the "drawing pose" goes to the first pose of the new animation. If the animation to set is the same one that is in progress, the "drawing pose" will not go to the first pose of the animation.

Sets the animation side.

Parameters
APointer to the Animator.
sideGU_SIDE_RIGHT, GU_SIDE_LEFT.
Note
You can use GU_SIDE_LEFT when you are sure that you have added the poses of the left animation, since in case they do not exist you run the risk of making readings in unknown sectors of the memory.

Update and draw the "drawing pose" according to animation that is running. For each character, invoke this function once per frame.

Parameters
APointer to the Animator.
speedposes per frame. Examples: 0.0f (stoped), 1.0f (1 pose/frame).
xx-coordinate (pixels) of the drawing.
yy-coordinate (pixels) of the drawing.
wwidth (pixels) of the drawing.
hheight (pixels) of the drawing.
useAlphaGU_TRANSPARENCY_ON, GU_TRANSPARENCY_OFF.

When the world context is established (guEnableWorld()) the coordinates will be taken on the world axes, and the drawing can be viewed depending on the position of the camera. If the world context is not set or is disabled (guDisableWorld()), the position coordinates are taken on the axes of the screen.

237 //---------------------------------------------------------------------------
238 // TEST_x Register - Pixel Test Settings
239 // ATE - Alpha Test (0=off, 1=on)
240 // ATST - Alpha Test Method
241 // 0=NEVER: All pixels fail.
242 // 1=ALWAYS: All pixels pass.
243 // 2=LESS: Pixels with A less than AREF pass.
244 // 3=LEQUAL, 4=EQUAL, 5=GEQUAL, 6=GREATER, 7=NOTEQUAL
245 // AREF - Alpha value compared to.
246 // AFAIL - What to do when a pixel fails a test.
247 // 0=KEEP: Don't update anything.
248 // 1=FBONLY: Update frame buffer only.
249 // 2=ZBONLY: Update z-buffer only.
250 // 3=RGBONLY: Update only the frame buffer RGB.
251 // DATE - Destination Alpha Test (0=off, 1=on)
252 // DATM - DAT Mode (0=pass pixels whose destination alpha is 0)
253 // ZTE - Depth Test (0=off, 1=on)
254 // ZTST - Depth Test Method.
255 // 0=NEVER, 1=ALWAYS, 2=GEQUAL, 3=GREATER
256 //---------------------------------------------------------------------------
